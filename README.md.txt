Projeto de Blog Pessoal, utilizado como uma forma de avaliação da compreensão do conteúdo completo de todo o bootcamp da Generation.

Utilizado o Java com Spring no backend, TypeScript e React no frontend. 

Possui funções completas do CRUD, como criar temas e postagens, bem como alterá-las ou excluí-las. É também possível listar todas as postagens e todos os temas.
